# Molecular Weight Calculater(Python)

## Overview
This project calculates the molecular weight of chemical compounds using Python.it
supports multi-letter elements,nested brackets, hydrates,and CSV-based input.

## Features
- One-letter and two-letter  elements
- Nested round and square brackets
- Hydrated compounds (CuSO4.5H2O)
- Stack-based parsing logic
- CSV file input
- Clean scientific output

## Why This project ?
Coming from a chemistry background,this project demonstrates how chemical computation logic 
can be translated into programming using Python.

## MATLAB vs Python
MATLAB was used during academic coursework for numerical computation.
The same logic was reimplemented in Python due to its open-source 
nature,better string passing,and industry relevance.
